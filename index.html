Regras de ouro
→ Alta de preço com OI crescente + funding moderada + MVRV abaixo de 3 = tendência real financiada por capital novo. Continue posicionado.
→ Alta de preço com OI caindo + funding alta = short squeeze frágil. Não entre alavancado — não há combustível novo sustentando o movimento.

→ Fluxo sugerido: comece pelo MVRV (contexto macro) → OI (intensidade da alavancagem) → Funding (direção do excesso). Se os 3 apontam para o mesmo lado, o sinal é forte.
